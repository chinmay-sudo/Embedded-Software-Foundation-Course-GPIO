# Set 3

## Overview

This directory contains ten problems demonstrating advanced sequential LED patterns on the ATmega2560 microcontroller (Arduino Mega). All programs use direct register access without Arduino helper functions. Each problem includes a Wokwi simulation link.

## Files

- Source files: `ETALVIS_LED_Set_3_Problem_*.c`
- README: `Readme.md`

## Problems

All problems use Eight LEDs connected to port A bit 0 to bit 7.

1. Glow the LED's in the following order 3,7 delay 2,6 delay 1,5 delay 0,4 delay  
   Simulation: https://wokwi.com/projects/451151839990705153

2. Glow the LED's in the following order 0,4 delay 1,5 delay 2,6 delay 3,7 delay  
   Simulation: https://wokwi.com/projects/451152810262273025

3. Glow 0 to 7 LEDs with delay and 7 to 0 LEDs with delay  
   Simulation: https://wokwi.com/projects/451153302373701633

4. Glow 0,2,1,3,4,6,5,7 LEDs with delay  
   Simulation: https://wokwi.com/projects/451153677652780033

5. Glow the LED's in the following order 0,1 delay 6,7 delay 2,3 delay 4,5 delay  
   Simulation: https://wokwi.com/projects/451155219621632001

6. Glow the LED's in the following order 4,5 delay 2,3 delay 6,7 delay 0,1 delay  
   Simulation: https://wokwi.com/projects/451155782479442945

7. Glow the LED's in the following order 0,1,2 delay 5,6,7 delay 3,4 delay  
   Simulation: https://wokwi.com/projects/451156026968577025

8. Glow the LED's in the following order 3,4 delay 5,6,7 delay 0,1,2 delay  
   Simulation: https://wokwi.com/projects/451156793387500545

9. Glow the LED's in the following order 0 then 0,1 then 0,1,2... the 0,1,2,3,4,5,6,7 with delay  
   Simulation: https://wokwi.com/projects/451157159127686145

10. Glow the LED's in the following order 7 then 7,6 then 7,6,5... then 7,6,5,4,3,2,1,0, with delay  
    Simulation: https://wokwi.com/projects/451157433182469121

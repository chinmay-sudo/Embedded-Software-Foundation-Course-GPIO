# Set 8

## Overview

This directory contains ten problems demonstrating switch input and LED output control on the ATmega2560 microcontroller (Arduino Mega). All programs use direct register access without Arduino helper functions. Each problem includes a Wokwi simulation link.

## Files

- Source files: `ETALVIS_Switch_Set_8_Problem_*.c`
- README: `Readme.md`

## Problems

All problems use Eight LEDs on PORTA and Eight push button switches on PORTB.

1. Press the 0th switch, glow all LEDs  
   Simulation: https://wokwi.com/projects/451472084013277185

2. Press the 2nd switch, glow all LEDs  
   Simulation: https://wokwi.com/projects/451472246971433985

3. Press the 3rd switch, glow 0th LED, delay, 1st LED, delay, up to 7th delay  
   Simulation: https://wokwi.com/projects/451472574642024449

4. Press any switch, glow 7th LED, delay, 6th LED, delay, up to 0th delay  
   Simulation: https://wokwi.com/projects/451473362601846785

5. Press the 0th switch, glow even LEDs  
   Simulation: https://wokwi.com/projects/451473833260450817

6. Press the 7th switch, glow odd LEDs  
   Simulation: https://wokwi.com/projects/451474060258292737

7. Press the 0th switch, 0th LED should glow, Press the 1st switch, 1st LED should glow ..... ...up to 7th switch  
   Simulation: https://wokwi.com/projects/451474355523671041

8. Press the 0th switch, 7th LED should glow, Press the 1st switch, 6th LED should glow..... ...up to 7th switch  
   Simulation: https://wokwi.com/projects/451494489211858945

9. Press any switch 0th LED ON, delay, OFF, 1st LED ON, delay, OFF .... Up to 7th LED.  
   Simulation: https://wokwi.com/projects/451495641508861953

10. Press 1st switch. Glow 0th LED, 3rd LED, 4th LED, 7th LED.  
    Simulation: https://wokwi.com/projects/451496457232877569

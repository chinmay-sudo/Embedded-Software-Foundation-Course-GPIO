# Set 9

## Overview

This directory contains eight problems demonstrating switch-triggered LED sequences on the ATmega2560 microcontroller (Arduino Mega). All programs use direct register access without Arduino helper functions. Each problem includes a Wokwi simulation link.

## Files

- Source files: `ETALVIS_Switch_Set_9_Problem_*.c`
- README: `Readme.md`

## Problems

All problems use Eight LEDs on PORTA and Eight push button switches on PORTB.

1. Press the 2nd switch first glow 1st LED, delay, 2nd LED, delay, 5th LED, delay, 6th LED.  
   Simulation: https://wokwi.com/projects/451503773056907265

2. Press the 3rd switch, first glow 3rd LED, delay, 1st LED, delay, 7th LED, delay, 4th LED.  
   Simulation: https://wokwi.com/projects/451511809696310273

3. Press the 2nd switch glow 0th LED, delay, 1st LED, delay, 5th LED, delay, 6th LED.  
   Simulation: https://wokwi.com/projects/451512248524351489

4. Press 7th switch first glow 0th LED, delay, 1st LED, Delay, 4th LED, delay, 7th LED.  
   Simulation: https://wokwi.com/projects/451512618285830145

5. Press 3rd switch first glow 7th LED, delay, 6th LED, delay, 3rd LED, delay, 2nd LED.  
   Simulation: https://wokwi.com/projects/451512757389928449

6. Press 2nd switch first glow 3rd LED, delay, 5th LED, delay, 7th LED delay, 6th LED.  
   Simulation: https://wokwi.com/projects/451513004232080385

7. Press 7th switch first glow 5th LED, delay, 3rd LED, delay, 0th LED, delay, 1st LED.  
   Simulation: https://wokwi.com/projects/451513162144488449

8. Press 1st switch & Press 7th switch one by one: Glow 5th LED for 30 seconds  
   Simulation: https://wokwi.com/projects/451513504340994049

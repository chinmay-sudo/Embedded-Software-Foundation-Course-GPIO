## Overview

This module contains practice problems for learning switch input handling and LED output control using bare-metal GPIO programming on the ATmega2560 microcontroller (Arduino Mega). All programs are written in C using direct register manipulation (DDRx, PORTx, PINx) without Arduino library functions.

## What I Learned

- Reading digital input from push button switches
- Switch debouncing techniques
- Conditional LED control based on switch states
- Custom pattern generation triggered by switch inputs
- Multi-switch combination logic

- **Microcontroller**: ATmega2560 (Arduino Mega)
- **Language**: C (bare-metal)

- **Hardware**: Push button switches and LEDs

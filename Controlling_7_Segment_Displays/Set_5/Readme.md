# Set 5

## Overview

This directory contains ten problems demonstrating 7-segment display control on the ATmega2560 microcontroller (Arduino Mega). All programs use direct register access without Arduino helper functions. Each problem includes a Wokwi simulation link.

## Files

- Source files: `ETALVIS_7_Seg_Set_5_Problem_*.c`
- README: `Readme.md`

## Problems

All problems use a Common Cathode 7 Segment Display connected to Port A.

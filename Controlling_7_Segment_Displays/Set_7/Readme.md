# Set 7

## Overview

This directory contains ten problems demonstrating multi-digit 7-segment display control on the ATmega2560 microcontroller (Arduino Mega). All programs use direct register access without Arduino helper functions. Each problem includes a Wokwi simulation link.

## Files

- Source files: `ETALVIS_*_7_Seg_Set_7_Problem_*.c`
- README: `Readme.md`

## Problems

1. Connect a 7 Segment Display on PORTA. Connect a 7 Segment Display on PORTB. DISPLAY number 25  
   Simulation: https://wokwi.com/projects/451330241325687809

2. Connect a 7 Segment Display on PORTA. Connect a 7 Segment Display on PORTB. DISPLAY number "00" to "99" with the delay of 1 sec  
   Simulation: https://wokwi.com/projects/451330998917714945

3. Connect a 7 Segment Display on PORTA. Connect a 7 Segment Display on PORTB. DISPLAY number "0.0" to "9.9" with the delay of 0.5 sec. Increment by 0.1  
   Simulation: https://wokwi.com/projects/451331671833024513

4. Connect two 7 Segment Displays on Port A. Share the eight port pins pin to all the Displays. Connect the common pin of 7 Segment displays to Bit 0 & Bit 1 of Port B. DISPLAY number 25  
   Simulation: https://wokwi.com/projects/451332264560606209

5. Connect two 7 Segment Displays on Port A. Share the eight port pins pin to all the Displays. Connect the common pin of 7 Segment displays to Bit 0 & Bit 1 of Port B. DISPLAY number "00" to "99" with delay of 1 sec  
   Simulation: https://wokwi.com/projects/451335618671417345

6. Connect a 7 segment display on PORTA. Connect a 7 segment display on PORTB. Connect a 7 segment display on PORTF. Connect a 7 segment display on PORTK. Display number 2025  
   Simulation: https://wokwi.com/projects/451336955274532865

7. Connect a 7 segment display on PORTA. Connect a 7 segment display on PORTB. Connect a 7 segment display on PORTF. Connect a 7 segment display on PORTK. Display number "0000" TO "9999" with 0.1 sec delay  
   Simulation: https://wokwi.com/projects/451412438535911425

8. Connect four seven segment displays on PORTA. Share the eight port pins pin to all the displays. Connect the common pin of 7 Segment displays to Bit 0, Bit 1, Bit 2 and Bit3 of PORTB. Display number 2025  
   Simulation: https://wokwi.com/projects/451414337003917313

9. Connect four seven segment displays on PORTA. Share the eight port pins pin to all the displays. Connect the common pin of 7 Segment displays to Bit 0, Bit 1, Bit 2 and Bit3 of PORTB. Display number "0000" TO "9999" with 0.1 sec delay  
   Simulation: https://wokwi.com/projects/451419147140157441

10. Connect four seven segment displays on PORTA. Share the eight port pins pin to all the displays. Connect the common pin of 7 Segment displays to Bit 0, Bit 1, Bit 2 and Bit3 of PORTB. Display number "0.000" TO "999.9" with 0.1 sec delay. Increment by 0.1  
    Simulation: https://wokwi.com/projects/451419859628754945

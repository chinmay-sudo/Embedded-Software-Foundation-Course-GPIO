# Set 10

## Overview

This directory contains ten problems demonstrating 4x4 keypad matrix reading and quad 7-segment display output on the ATmega2560 microcontroller (Arduino Mega). All programs use direct register access without Arduino helper functions. Each problem includes a Wokwi simulation link.

## Files

- Source files: `ETALVIS_Keypad_Set_10_Problem_*.c`
- README: `Readme.md`

## Problems

All problems use a 4x4 Keypad and Quad 7 Segment Display.

1. Press any switch and display that number  
   Simulation: https://wokwi.com/projects/451636317432074241

2. Press 4 digits and display that number  
   Simulation: https://wokwi.com/projects/451651028208773121

3. Single digit addition with 2-digit output  
   Simulation: https://wokwi.com/projects/451665934935237633

4. Perform two digit addition  
   Simulation: https://wokwi.com/projects/451739139175557121

5. Perform three digit addition  
   Simulation: https://wokwi.com/projects/451789594375375873

6. Perform addition. If the total exceeds 9999 then display "EEEE"  
   Simulation: https://wokwi.com/projects/451842858703761409

7. Perform subtraction. If the total exceeds 9999 then display "EEEE". Display negative results also  
   Simulation: https://wokwi.com/projects/451844176425263105

8. Perform multiplication. If the total exceeds 9999 then display "EEEE"  
   Simulation: https://wokwi.com/projects/451846711275216897

9. Perform division. Display only quotient  
   Simulation: https://wokwi.com/projects/451863856209096705

10. Perform division. Display only remainder  
    Simulation: https://wokwi.com/projects/451864888084488193
